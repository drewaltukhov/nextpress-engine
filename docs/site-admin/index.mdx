The site-admin section covers the housekeeping tasks every NextPress site eventually needs: keeping search engines happy, redirecting old URLs after a rearrangement, and snapshotting the database so you can roll back.

None of it requires opening a code editor — every screen mentioned here lives under **Admin** in the top-right user menu, or under the section it belongs to in the admin sidebar.

## What's in here

| Page | What it covers |
| --- | --- |
| [SEO](/docs/site-admin/seo) | Site title and tagline, sitemap / robots / RSS, OG defaults, structured-data toggles |
| [Redirects](/docs/site-admin/redirects) | Adding 301/302 redirects, importing CSV lists, the on-edit auto-redirect for slug changes |
| [Backups & restore](/docs/site-admin/backup-restore) | Exporting the database, importing a backup, what's included / excluded |
